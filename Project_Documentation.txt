IT Helpdesk Ticketing System – Project Documentation
1. Project Title
IT Helpdesk Ticketing System using Jira

2. Objective
To simulate how IT support teams create, track, and resolve tickets using Agile Scrum in Jira.

3. Tools Used
Jira Software (Team-Managed)
Scrum Board
Issue Types: Epic, Story, Task, Bug, Sub-task

4. Work Done
Created a Scrum-based project in Jira
Added Epic, User Stories, Tasks, Subtasks, and Bugs
Created Sprint and moved issues from Backlog → To-Do → In-Progress → Done
Created Dashboard and charts (Created vs Resolved, Issue statistics)
